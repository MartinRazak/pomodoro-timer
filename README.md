# Pomodoro Timer

A simple Pomodoro timer built with Python.  
It helps improve focus and productivity by using timed work sessions and breaks based on the Pomodoro technique.

---

## My program

I have made 2 versions, one CLI to get things started. And then I experimented with the tkinter and playsound libraries for the first time to make a GUI and use it as my everyday pomodoro windows application.

---

## Features

- 25-minute work sessions (customizable)
  Short breaks between sessions
- Long break after 4 Pomodoro cycles
- Automatic cycle looping
- Updated CLI formatting

---

## How it works

The Pomodoro technique alternates between focused work and rest:

1. Work session
2. Short break
3. Repeat
4. After 4 cycles → long break
